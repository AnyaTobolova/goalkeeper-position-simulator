# Сценарии и правильные решения

Документ фиксирует текущие игровые сценарии симулятора вратаря, принципы оценки и оставшиеся идеи для будущих режимов.

## 1. Общая логика правильной позиции

Базовые правила:

1. Вратарь стоит на линии от мяча к середине ворот или рядом с ней.
2. Вратарь не остается в воротах и не стоит за линией ворот, если ситуация требует выхода.
3. При боковом мяче сначала контролируется ближний угол.
4. Глубина зависит от расстояния до мяча: дальний удар - можно стоять выше, близкий удар - нельзя выбегать слишком далеко.
5. Корпус должен быть развернут к мячу.

Особые правила:

1. При высоком навесе вратарь не играет как на низкий прострел: он держит ворота, траекторию и мяч под перекладиной.
2. При угловом на открытом поле вратарь занимает дальнюю стойку, чтобы видеть поле и траекторию.
3. При угловом в зале/мини-футболе вратарь держит середину ворот.
4. При штрафном стенка закрывает ближний угол, а вратарь занимает середину открытой части ворот.

## 3. Что осталось за пределами текущего режима

Это не критичные пробелы для позиционного симулятора, а отдельные будущие режимы:

1. Коммуникация с защитниками: выбрать команду защитнику, например "держи дальнюю", "закрой пас", "подними линию".
2. Начало атаки после ловли мяча: ввод рукой, пас защитнику, вынос, решение под прессингом.
3. Игра на выходах руками в борьбе: ловить, отбивать кулаком или остаться. Это уже отдельная механика тайминга, не только постановки.

## 4. Универсальная оценка сценариев

Сценарии не должны иметь уникальный ручной разбор для каждого уровня. Уровень задает учебную ситуацию: мяч, игроков, стартовую позицию вратаря, возможную стенку и при необходимости тренерские допуски. Оценка и обратная связь собираются универсально.

Для каждого сценария определяется `scenarioType`:

| Тип сценария | Основные критерии |
| --- | --- |
| `central_shot` | линия, глубина, угол удара, готовность |
| `side_shot` | линия, глубина, ближний угол, готовность |
| `sharp_angle` | линия, глубина, ближний угол, дальний угол |
| `long_shot` | линия, глубина, готовность, риск переброса |
| `close_shot` | линия, глубина, готовность, риск переброса |
| `one_v_one` | линия, глубина, контроль выхода, готовность |
| `one_v_one_loose_touch` | линия, глубина, контроль выхода, готовность |
| `pass_or_cutback` | перестроение, новая линия мяча, глубина, готовность |
| `cross_goal` | перестроение, ближний угол, дальний угол, готовность |
| `high_cross` | траектория, обзор мяча, глубина, контроль ворот |
| `corner` | траектория, обзор мяча, контроль ворот, стартовая позиция |
| `free_kick` | стенка, обзор мяча, открытая часть ворот, глубина |
| `defender_pressure` | линия, глубина, помощь защитника, готовность |
| `sweeper_position` | глубина, зона за спиной, линия, готовность |

Главная ошибка определяется как `errorType`: например `TOO_HIGH`, `TOO_DEEP`, `TOO_LEFT`, `TOO_RIGHT`, `NEAR_POST_OPEN`, `NOT_ADJUSTED_AFTER_PASS`, `NO_BALL_VISIBILITY`, `WALL_COUNT_WRONG`, `WALL_POSITION_WRONG`.

Правило согласованности:

- если критерий красный, его текст объясняет проблему;
- если критерий зеленый, его текст не содержит опасности;
- если критерий красный, нельзя использовать формулировки, которые звучат как правильное действие: `угол сокращен`, `можешь реагировать`, `безопасно`, `закрыт`;
- если совет говорит сместиться левее или правее, критерий `Линия к мячу` тоже должен требовать поправки;
- если мяч по центру, используется `Угол удара`, а не `Ближний угол`;
- основной совет всегда один.

Правило зон:

- точка ног внутри зеленой зоны дает `Отлично`, если не нарушены отдельные обязательные условия сценария вроде стенки при штрафном;
- точка ног внутри оранжевой зоны считается допустимой позицией: это может быть `Почти` или `Нужно поправить`, но не `Опасно`;
- если точка ног немного вышла за оранжевую зону, результат остается в буферной зоне `Нужно поправить`;
- `Опасно` по глубине ставится только при явном попадании точки ног в красную зону;
- если красной зоны глубины на поле нет, статус `Опасно` нельзя объяснять глубиной.

## 5. Визуальные подсказки на поле

Визуал должен соответствовать критериям справа.
Легенда обозначений показывается рядом со статусом результата, а не поверх поля, чтобы не закрывать игроков, мяч и зоны.

| Элемент | Когда показывается | Что означает |
| --- | --- | --- |
| белый пунктир | после ответа и в разборе | линия удара от мяча к центру ворот |
| зеленая капсула | после ответа | лучшая позиция |
| оранжевая капсула | после ответа | допустимо, но не идеально |
| красная зона | в режиме `Показать почему`, только при явной ошибке глубины | опасная глубина: слишком близко к воротам или слишком далеко вышел |
| желтая стрелка | только при ошибке или `Почти` | одно действие для исправления |
| голубой пунктир | только при пасе, обзоре или навесе | движение мяча, обзор мяча или траектория |
| стенка/зона стенки | при штрафном | что закрывает стенка |

Голубой пунктир нельзя использовать как декоративную линию или как ось правильной позиции.

Связь критериев и подсказок:

- `Перестроение` включает старую точку мяча, новую точку мяча и путь движения мяча;
- `Обзор мяча` включает голубую линию видимости;
- `Глубина: слишком далеко` включает красную зону `TOO_HIGH_ZONE`;
- `Глубина: слишком глубоко` включает красную зону `TOO_DEEP_ZONE`;
- красная зона глубины используется только для `TOO_DEEP`, `TOO_HIGH`, `PASSIVE_1V1`, `RUSHED_1V1`;
- ошибки линии к мячу, ближнего угла, стенки, обзора мяча и перестроения после паса не включают красную зону глубины;
- ошибка по линии объясняется белой линией удара, границами угла к штангам и стрелкой к ближайшей правильной точке;
- ошибка ближнего угла включает сектор удара;
- штрафной включает стенку и обзор мяча.

## 6. Каталог текущих сценариев

### Угол удара и ближний угол

| Сценарий | Правильное решение | Почему так |
|---|---|---|
| Мяч справа у штрафной | Сместиться вправо по линии мяча, выйти из ворот, корпус к мячу. | Ближний угол справа должен быть закрыт первым. |
| Острый угол слева | Сместиться к левой ближней штанге, не стоять по центру. | С острого угла опаснее простой удар в ближний угол. |
| Широко справа | Сильно сместиться вправо, но не выбегать чрезмерно. | Чем шире мяч, тем важнее ближняя штанга. |
| Мяч слева у штрафной | Сместиться влево и закрыть левый ближний угол. | Центральная позиция открывает ближнюю штангу. |
| Удар по центру | Держать центральную линию ворот. | Когда мяч по центру, уход в сторону открывает половину ворот. |

### Глубина позиции

| Сценарий | Правильное решение | Почему так |
|---|---|---|
| Дальний удар по центру | Выйти выше линии ворот, держать центр. | Чем дальше мяч, тем больше времени на реакцию и тем сильнее можно уменьшить угол. |
| Близкий удар по центру | Не выбегать слишком далеко, остановиться перед ударом. | При близком мяче слишком высокий выход дает шанс на обводку или переброс. |
| Дальний удар справа | Выйти выше и сместиться вправо по линии мяча. | Нужно одновременно уменьшить ворота и закрыть угол. |
| Дальний удар слева | Выйти выше и сместиться влево. | Та же логика, зеркально. |
| Близкий удар справа | Быть готовым к удару, не перелетать вперед. | Вратарь должен сократить угол, но сохранить возможность реакции. |

### 1-в-1

| Сценарий | Правильное решение | Почему так |
|---|---|---|
| Выход 1-в-1 по центру | Выйти навстречу, сократить угол, остановиться перед ударом. | Вратарь делает ворота меньше, но не бросается без контроля. |
| Два нападающих по центру | Закрыть игрока с мячом, но не потерять готовность к пасу. | Главная угроза - мяч, но второй игрок меняет риск передачи. |
| 1-в-1 справа | Выйти к игроку с мячом и закрыть правый угол. | Пассивность на линии оставляет нападающему много ворот. |
| 1-в-1 слева | Выйти к мячу и сместиться влево. | Зеркальный вариант правого 1-в-1. |
| 1-в-1, мяч отпущен вперед | Выйти смелее, пока мяч далеко от ноги нападающего. | Когда мяч отпущен, у вратаря есть момент сократить дистанцию. |

### Защитник и перегруз

| Сценарий | Правильное решение | Почему так |
|---|---|---|
| Защитник давит справа | Не выбегать слишком рано, держать ворота и угол. | Если защитник уже мешает, ранний выход может открыть ворота. |
| Три игрока перед штрафной | Держать равновесную позицию на линии мяча. | Есть удар и несколько вариантов паса, поэтому нельзя дергаться без причины. |
| Защитник проигран | Играть активнее как в 1-в-1. | Если защитник сзади, вратарь становится главным защитником пространства. |
| Защитник перекрывает удар | Не выходить резко, держать ворота. | Защитник уже снижает качество удара, задача вратаря - не открыть лишнее. |
| Трое атакуют слева | Сместиться к мячу, контролировать ближний угол и пас в центр. | Оставаться по центру при атаке слева опасно. |
| Трое атакуют справа | Сместиться вправо, держать ближнюю штангу и пас перед собой. | Зеркальный вариант атаки слева. |
| Перегруз перед воротами | Вернуться ближе к центру линии мяча. | При центральной угрозе уход в сторону открывает ворота. |

### Пасы, прострелы, откаты

| Сценарий | Правильное решение | Почему так |
|---|---|---|
| Пас слева в центр | Перестроиться из старой левой позиции к новой линии мяча по центру. | После паса старая позиция больше не защищает ворота. |
| Прострел справа | Сначала закрыть ближнюю штангу, быть готовым к мячу вдоль ворот. | При простреле нельзя открывать ближний угол. |
| 2-в-1, пас налево | Быстро сместиться к новой угрозе слева. | Мяч поменял сторону, значит меняется и позиция вратаря. |
| Пас справа в центр | Вернуться к центральной линии мяча. | Пас в центр убирает прежнюю фланговую позицию. |
| Откат с левого фланга | Не оставаться у ближней штанги, сместиться под удар из центра. | Откат назад часто дает удар с лучшего угла. |
| Откат с правого фланга | Сместиться от правой штанги к новой точке мяча. | Новая позиция мяча важнее старого фланга. |
| Передача на дальнюю штангу | Успеть сместиться к новой стороне и не остаться на старой штанге. | Дальняя штанга становится новой зоной угрозы. |
| Поздний пас направо | Сместиться вправо после поздней передачи. | Вратарь должен реагировать на последнюю точку мяча. |
| Поздний пас налево | Сместиться влево после поздней передачи. | Зеркальный вариант позднего паса направо. |

### Высокие навесы

| Сценарий | Правильное решение | Почему так |
|---|---|---|
| Навес справа под перекладину | Держать ворота, не выбегать высоко, корпус к мячу. | Мяч летит дугой и может опуститься под перекладину. |
| Навес слева под перекладину | Оставаться в рабочей позиции в воротах, контролировать траекторию. | Высокий навес не играется как низкий прострел. |
| Навес на дальнюю штангу | Сместиться к дальней штанге, но не уходить за линию ворот. | Дальняя штанга становится главной зоной угрозы. |
| Навес в толпу игроков | Держать рабочую позицию и не вылетать вслепую. | Если траектория спорная и впереди много игроков, пустые ворота опаснее. |

### Угловые

| Сценарий | Правильное решение | Почему так |
|---|---|---|
| Угловой справа | Контролировать ближнюю зону и не провалиться в ворота. | Это тренирует выход из ворот и контроль линии. |
| Угловой слева | Занять позицию у ближней зоны и развернуться к мячу. | Зеркальный вариант углового справа. |
| Угловой справа: открытое поле | Стартовать в дальней игровой зоне, видеть мяч и не проваливаться за линию ворот. | Это не универсальное правило для любого углового: позиция зависит от подачи и обзора. |
| Угловой слева: открытое поле | Стартовать в дальней игровой зоне справа, сохраняя контроль ворот и обзор мяча. | Зеркальный вариант конкретной подачи слева, а не механическое правило “всегда к стойке”. |
| Угловой справа: зал | Держать середину ворот. | В зале подача/розыгрыш быстрее, стойка не должна уводить из ворот. |
| Угловой слева: зал | Держать середину ворот. | Та же логика, зеркально. |

### Штрафные со стенкой

| Сценарий | Правильное решение | Почему так |
|---|---|---|
| Штрафной справа | 3 игрока в стенке, стенка закрывает ближний угол, вратарь в открытой части ворот. | Стенка и вратарь делят ворота на зоны ответственности. |
| Штрафной слева | 3 игрока, стенка закрывает левый ближний угол, вратарь смещается в открытую часть. | Зеркальный вариант штрафного справа. |
| Штрафной по центру | 4 игрока, плотная стенка перед мячом, вратарь не прячется за стенкой. | Центральный удар опаснее по обеим сторонам. |
| Близкий штрафной по центру | 4 игрока, плотная стенка, вратарь не выходит далеко. | Чем ближе мяч, тем меньше времени на реакцию. |
| Очень боковой штрафной справа | 2 игрока в стенке, компактная стенка закрывает ближнюю траекторию. | С острого штрафного большая стенка забирает лишних защитников. |
| Штрафной с розыгрышем | 3 игрока в стенке, вратарь видит мяч и возможный пас. | Нужно закрыть удар, но не застыть за стенкой. |
| Штрафной на маленькие ворота | 2 игрока в стенке, вратарь сохраняет обзор. | На маленьком поле слишком большая стенка мешает защите и обзору. |

## 7. Вывод

Для позиционного тренажера текущий набор закрывает все обязательные сценарии из обсуждения: удары, глубину, 1-в-1, пасовые перестроения, перегрузы, навесы, угловые двух типов и штрафные со стенкой.

Следующие крупные расширения лучше делать отдельными режимами: коммуникация с защитниками и начало атаки после ловли мяча.
